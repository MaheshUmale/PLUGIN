4. Enterprise Sideload & Deployment Guide

1

Extract / Package

Save all compiled files above into a local directory on your machine named mrm-compliance-assistant.

2

Open Developer Mode

Launch Google Chrome, go to chrome://extensions, and toggle the "Developer mode" switch in the top right corner.

3

Load Unpacked

Click on the "Load unpacked" button in the top left, and select your extracted folder containing the extension files.

4

Pin & Enforce

Pin the shield icon 🛡️ to your browser bar. The extension will automatically run in the background, enforcing RBI MRMF rules on all whitelisted banking domains.
